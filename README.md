TRACEA APP

A modern SaaS platform built for artisans to streamline intervention management, documentation, and automated PDF evidence generation.
🚀 Tech Stack

    Frontend: React 18 + TypeScript

    Build Tool: Vite

    Styling: Tailwind CSS

    UI Components: Shadcn UI (Radix UI primitives)

    Forms & Validation: React Hook Form + Zod

    Data Fetching: TanStack Query

    Backend integration: Supabase

🛠️ Getting Started
Prerequisites

    Node.js (v18.0.0 or higher)

    npm or bun

Installation

    Clone the repository:
    Bash

    git clone <your-repo-url>
    cd <your-project-folder>

    Install dependencies:
    Bash

    npm install

    Run the development server:
    Bash

    npm run dev

📁 Project Structure
Directory	Description
src/components	UI components and Shadcn primitives
src/hooks	Custom React hooks for logic and state
src/lib	Configuration files (Supabase, utils, etc.)
src/pages	Application views and routing
src/types	TypeScript interfaces and definitions
📜 Available Scripts

    npm run dev - Start the development server.

    npm run build - Build the project for production.

    npm run lint - Run ESLint for code quality checks.

    npm run preview - Preview the production build locally.
